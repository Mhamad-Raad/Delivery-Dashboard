Migration steps — 20260420xxxxxx_ReplaceCategorySystem

  1. 1Create VendorCategories table + unique index on Name. ADD
  2. 2Seed 6 rows with explicit ids matching VendorType enum values (Restaurant=1, Cafe=2, Market=3, Bakery=4, Pharmacy=5, Other=6). SEED
  3. 3Add nullable VendorCategoryId column to Vendors. ADD
  4. 4Backfill: UPDATE Vendors SET VendorCategoryId = CAST(Type AS INTEGER). BACKFILL
  5. 5Make Vendors.VendorCategoryId NOT NULL; add FK → VendorCategories.Id (OnDelete Restrict). CONSTRAIN
  6. 6Drop Vendors.Type column. DROP
  7. 7Rename table Catagory → Category. RENAME
  8. 8Drop Category.ParentCategoryId column + self-ref FK. DROP
  9. 9Null out Product.CategoryId for all products, then delete all existing Category rows. Dev DB only (approved). WIPE
  10. 10Add nullable VendorId; make NOT NULL; FK → Vendors.Id (OnDelete Cascade). ADD
  11. 11Add unique index on (VendorId, Name). INDEX
